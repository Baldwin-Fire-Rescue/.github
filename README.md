# .github

Default [community health files](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file) and profile page for [Baldwin Fire Rescue](https://github.com/Baldwin-Fire-Rescue).

Files in this repository are automatically inherited by all repositories
in the organization that don't provide their own copy:

- **CODE_OF_CONDUCT.md** — Contributor Covenant v2.1
- **CONTRIBUTING.md** — contribution guidelines and PR checklist
- **SECURITY.md** — vulnerability reporting policy
- **FUNDING.yml** — sponsor and donation links
- **Issue & PR templates** — bug reports, feature requests, PR checklist

The `profile/README.md` powers the organization profile page shown at
[github.com/Baldwin-Fire-Rescue](https://github.com/Baldwin-Fire-Rescue).

## Overriding defaults

Any repository can override these files by placing its own copy at the
same path. For example, a repo with a project-specific `CONTRIBUTING.md`
will use its own version instead of the default here.
